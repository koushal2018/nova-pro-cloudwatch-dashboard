# Product Overview

## Nova Pro CloudWatch Dashboard

A CloudFormation-based monitoring solution for enterprises using Amazon Bedrock's Nova Pro foundation model. Provides comprehensive observability for AI workload operations through pre-configured CloudWatch dashboards.

## Key Features

- One-click deployment via CloudFormation template
- Real-time monitoring of usage patterns, tokens per minute (TPM), costs, and latency
- Pre-configured alarms for error rates, throttling, latency breaches, and cost limits
- Guardrail intervention tracking for content safety compliance
- Cost tracking with cache efficiency metrics
- SNS integration for operational alerts

## Target Users

- Enterprise administrators deploying AI infrastructure
- Operations engineers monitoring model performance
- Financial controllers tracking AI costs
- Reliability engineers ensuring SLA compliance
- DevOps teams managing Bedrock workloads
- Compliance officers monitoring content safety
